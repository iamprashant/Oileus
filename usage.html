Using Oileus is very easy, just we need to add some attribute to our HTML tags. Let's see how-

Oileus forms

Oileus forms are special forms, whose request sent by oileus. All validation code can reside at client or server side also. Follwoing is an example of writing an Oileus form-
<form action="http://www.codingdash.com/signup" os-through="oileus" os-class="Account" os-event="preSignup" os-handler="onSignup"> <input type="text" name="name" id="name"> <input type="text" name="city" id="city"> <input type="password" name="password" id="password"> <input type="submit"> </form>
And the Oileus controller for the same class can be written as following-
/** * For account activity * @constructor */ oileus.controller("Account", function() { //constructor code }) /** * Will run before the request will be sent. Simply return false to abort the * request */ .actFor("Account", "preSignup", function(el) { //code return true; }) /** * Will run after the response. * request */ .actFor("Account", "onSignup", function(info, el) { //code });

Oileus links

In the way how forms are using Oileus, anchor tags also can use the Oileus. For example-

<a href="http://www.codingdash.com/increase/like" os-through="oileus" os-class="MyController" os-event="myEvent" os-handler="myHandler"> Like </a>

Oileus Active

When you want to perform some action, when specific contents gets ready. In such case you can use Oileus active. Here is an example for how to use it-

<div os-rel = "activate" os-class = "MyController" os-active = "myActiveAction"> ... </div>

Oileus Target

It binds the target to any kind of element other than anchor tag. It's not a good practice but some situation may demand this. Here is an example for how to use it-

<div os-rel="target" href="http://codingdash.com"> ... </div>

Autoload

Loads the content dynamically of given URL. Here is the example-

<div os-through = "oileus" src = "http://www.codingdash.com/about/me"> ... </div>

This will load the content from the given URL. This feature can be used only divisions.
